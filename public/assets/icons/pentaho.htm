SFTP To Go

SFTP To Go is a fully managed secure, scalable and reliable cloud file storage with support for the SFTP, FTPS, S3 and HTTPS protocols. It allows to share and integrate with 3rd parties using popular and secure protocols and automate data management and processing using APIs and webhooks.

How does it work?

Use Pentaho to read, process and write data to or from SFTP To Go.

  1. Create a new job in Pentaho Data Integration.
  2. Add either Get a file with SFTP to pull files from SFTP To Go, or Put a file with SFTP to upload files to SFTP To Go.
  3. In SFTP To Go, create or select the credential Pentaho will use, then copy the host, username, password, and use port 22.
  4. Enter those details in Pentaho and set the remote and local paths.
  5. Run the job and confirm Pentaho can read from or write to the selected SFTP To Go folder.

For further reading, see Pentaho job entries reference and connect using SFTP.
